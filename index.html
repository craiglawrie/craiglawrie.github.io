About

I have a degree in physics from Case Western Reserve University and more than eight years of diverse engineering experience. I have designed MRI coils to speed up medical scans and improve the image quality, and I have developed helicopter driveshafts to reduce aircraft weight. My product development has extended from concept through production and included generating build and test instructions and quality inspection documentation. During this time, I programmed in Microsoft VBA, MATLAB, QTScript, ANSYS APDL, and more.

My particular strength lies in my analytical ability, especially in identifying and combining the important details for the target result. This skill has contributed to my success in physics, engineering, programming, and, perhaps surprisingly, teaching ballroom dance. I am attending Tech Elevator Cleveland to develop my skills in C#, ASP.NET MVC, HTML, CSS, Javascript, and SQL, and I am using any extra time to study Unity and Python. I am looking for opportunities where I have a steady supply of intellectually stimulating tasks.

Countdown Timer

A simple console program written in C#. This project started as an investigation into querying the system clock. The System.Timer class worked even better, allowing the use of an event handler to avoid tying up system resources. I was able to tidy up the code significantly by using an Actions array to grab individual ASCII art numbers. See it on github.

Time at Temperature

This project is a data processing excercise used to predict the cure time of epoxies and other thermosetting plastics. Data containing the time, temperature, and energy released by the sample was processed in a Python script following algorithms described by Sbirrazzuoli and Vyazovkin (see references). See in on github.
